######目录说明

#####资源目录（主要针对图片）（美术增加目录）
Common   模板套装公共资源目录
GameRes  模版套装非公共资源
Special  模版套装中资源（必换、要求数量较大)(主要是背景和人物)


####脚本目录（导出的.lua 与相关的资源）
LayOut 模版套装底板（布局模型）已经导出可以使用的
UI    cocos ccs 工程 用于导出模版套装
